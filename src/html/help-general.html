General

The map saves automatically after every edit. Images used as backgrounds are stored in the interactive-map folder. You should copy that folder, if you plan to move maps to other workspaces.

Holding Shift and selecting an area by dragging allows you to zoom into that area.

You can use html tags inside descriptions. The styled text will be visible in the popup windows of clicked features.

Bootstrap Icons

Bootstrap icons can be embedded into descriptions, use @(name-of-the-icon).
For example:
@(bi-patch-question-fill) results in @(bi-patch-question-fill).

Dendron

You can use Dendron's note references to access the content of notes.

Note Reference ![[note]] Entire content
Block Reference ![[note#block-name]] Entire block
Block Range Reference ![[note#block-start:#block-end]] All blocks including and between the starting and ending
